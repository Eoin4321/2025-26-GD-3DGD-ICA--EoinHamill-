# EscapeRoomGame
A github to store my escape room project.

## Overview

This project is an escape room game created in Unreal Engine for my ICA assignment. The player explores an apartment building while attempting to steal valuables hidden inside. The level begins in an apartment lobby and gradually leads into a bedroom, where the main puzzle sequence and final objective are located.

The game is set late at night and uses a dark, moody atmosphere. The apartment is designed to feel old, neglected, and slightly unsettling. Crooked paintings, knocked-over furniture, and scattered objects are used to suggest that the space has been disturbed before the player arrives.
